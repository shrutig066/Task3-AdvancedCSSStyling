# 🌐 TechNova - Responsive Business Website

A modern and responsive business website developed using **HTML5, CSS3, Bootstrap 5, and JavaScript**. This project was created as part of the **Cognifyz Technologies Web Development Internship – Level 2 Task 3**.

## 🚀 Live Features

- Responsive Navigation Bar
- Hero Section
- About Us Section
- Services Section
- Animated Statistics Counter
- Portfolio Gallery
- Testimonials Carousel
- Contact Form with Validation
- Responsive Footer
- Smooth Scrolling Navigation
- Hover Effects & CSS Animations
- Mobile-Friendly Design
- Back-to-Top Button

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome
- Google Fonts
